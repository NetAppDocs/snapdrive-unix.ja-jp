= LUN 上に作成されたファイルシステムをホストから切断するためのコマンド構文
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
lun storage disconnect コマンドを使用 SnapDrive して 'LUN 上で直接作成されたファイル・システムを削除するには ' 次の構文を使用します

SnapDrive storage disconnect -fs file_spec [-fstype type] [-vmtype type]
