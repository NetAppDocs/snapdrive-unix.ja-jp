= Snapshot コピーを切断しています
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
LUN 、 LUN 上のファイルシステム、ディスクグループ、 NFS ディレクトリ、または LUN 上の共有ディスクグループ、ホストボリューム、およびファイルシステムから、 Snapshot コピーを切断できます。


NOTE: SnapDrive for UNIX 4.2 以降のバージョンでは、スプリットクローンボリュームを切断できます。
