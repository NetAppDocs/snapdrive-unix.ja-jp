= SnapDrive for UNIX でのロールベースアクセス制御の設定
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX の Role-Based Access Control （ RBAC ；ロールベースアクセス制御）を設定するには、さまざまなタスクを完了する必要があります。このタスクは、 Operations Manager コンソールまたはコマンドラインインターフェイスを使用して実行できます。
