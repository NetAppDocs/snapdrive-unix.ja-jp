= ストレージシステムのリストの表示
:icons: font
:imagesdir: ../media/


[role="lead"]
ホスト・システムで SnapDrive config list コマンドを使用すると、すべてのストレージ・システムを表示できます。

.手順
. 次のコマンドを入力します。
+
SnapDrive 構成リスト

+
[listing]
----
snapdrive config list
username appliance name appliance type
-----------------------------------------------------
root      roaster     StorageSystem
----
