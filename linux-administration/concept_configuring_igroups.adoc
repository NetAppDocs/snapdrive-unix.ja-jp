= igroup を設定しています
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX では、ストレージシステム上に存在する igroup を設定できます。設定された igroup を使用して、ストレージシステム上の LUN がマッピングされます。


NOTE: igroup には、ソースイニシエータとデスティネーションイニシエータが含まれている必要があります。
