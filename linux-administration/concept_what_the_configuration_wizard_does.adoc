= SnapDrive 設定ウィザードの機能
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive 設定ウィザードでは、環境に応じて、 SnapDrive for UNIX および NFS または SAN を設定できます。また 'opendrive.conf ファイルを開き ' 構成変数を編集することもできます
