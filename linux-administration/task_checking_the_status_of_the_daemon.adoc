= デーモンのステータスを確認しています
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
デーモンのステータスをチェックして、デーモンが実行されているかどうかを確認できます。デーモンがすでに実行されている場合は、 SnapDrive for UNIX 構成ファイルが更新されるまで、デーモンを再起動する必要はありません。

root ユーザとしてログインする必要があります。

.手順
. デーモンのステータスを確認します。
+
「 * snapdrived status * 」を入力します
