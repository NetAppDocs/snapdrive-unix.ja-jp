:icons: font
:imagesdir: ../media/

移行したストレージ・システムがホスト・システムに含まれているかどうかを確認するには、 SnapDrive config migrate list' コマンドを実行します。

.手順
. 次のコマンドを入力します。
+
'*SnapDrive config migrate list*
