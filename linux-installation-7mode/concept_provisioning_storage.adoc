= ストレージのプロビジョニング
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX では、ホストからコマンドラインオプションを使用して、ストレージをプロビジョニングし、既存のストレージシステムに接続できます。Data ONTAP を使用してストレージをプロビジョニングすることもできます。
