SnapDrive storage resize コマンドを使用して、エンティティのサイズを縮小することはできません。サイズを拡張するには、このコマンドを使用する必要があります。

* バイトをストレージに追加する (`-growby-`)
* サイズを指定したバイト・サイズに増やす（「 -growto 」）。
