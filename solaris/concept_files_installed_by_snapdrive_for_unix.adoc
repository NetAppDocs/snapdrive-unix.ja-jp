= SnapDrive for UNIX によってインストールされるファイル
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX では、実行ファイル、構成ファイル、アンインストールファイル、診断ファイル、マニュアルページなど、複数のファイルがインストールされます。 ホスト。これらのファイルを使用すると、問題を検出し、現在のバージョンの製品の構成情報を SnapDrive for UNIX に格納できます。
