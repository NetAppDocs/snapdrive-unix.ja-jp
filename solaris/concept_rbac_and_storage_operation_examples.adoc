= RBAC とストレージ処理の例
:icons: font
:imagesdir: ../media/


[role="lead"]
ロールベースアクセス制御を使用すると、割り当てられた機能に応じてストレージの処理を実行できます。ストレージ操作を実行するための適切な機能がない場合は、エラーメッセージが表示されます。
