= Snapshot コピーの名前変更
:icons: font
:imagesdir: ../media/


[role="lead"]
「 snapshot-rename 」コマンドを使用すると、 Snapshot コピーの名前を変更できます。複数のストレージシステムまたはストレージシステムボリュームにまたがる Snapshot コピーの名前も変更できます。
