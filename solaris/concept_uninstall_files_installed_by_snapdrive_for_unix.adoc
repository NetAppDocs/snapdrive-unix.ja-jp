= SnapDrive for UNIX によってインストールされたファイルをアンインストールします

:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX では、アンインストールファイルを使用して SnapDrive for UNIX が削除されます。

ファイルは次の場所にインストールされます。

/opt/NTAPsnapdrive/snapdrive.conf
