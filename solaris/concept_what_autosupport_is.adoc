= AutoSupport とは
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
AutoSupport を使用すると、 SnapDrive for UNIX は、 SnapDrive によって発生したすべての動作障害を、「 /etc/log/ems 」ファイルに含まれるストレージ・システムの EMS ログ・ビューに送信できます。
