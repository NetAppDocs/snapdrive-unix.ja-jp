= トレースログファイルとは
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
テクニカルサポートは、トレースログファイルを使用して問題のトラブルシューティングを行います。

トレースログファイルを有効にしても、システムパフォーマンスには影響しません。デフォルトでは、このファイルは有効になっています。「 napdrive.conf trace -enabled 」変数を「 off 」に設定すると、無効にできます。
